# Task 2 – Student Digital Portfolio

## Platforms Set Up

### 1. GitHub (github.com)
GitHub is the world's largest platform for hosting and sharing code. It uses a version control system called Git, which tracks every change made to a project. For students, GitHub serves as a living portfolio of technical work — every project, script, or assignment uploaded here is visible to future employers and collaborators. I created a profile README repository (a special repository named after my username) containing my name, branch, year, and a sentence about my learning goals.

### 2. LinkedIn (linkedin.com)
LinkedIn is the world's leading professional networking platform, used by recruiters, companies, and professionals across all industries. I updated my Education section with my current B.Tech programme, VIT Bhopal University, and my expected graduation year. Over time, this profile will grow to include internships, projects, certifications, and recommendations.

### 3. Kaggle (kaggle.com)
Kaggle is a platform owned by Google, primarily used for data science and machine learning. It hosts datasets, competitions, and free courses. Even as a first-year student, setting up a Kaggle account is useful because it gives access to beginner-friendly notebooks and datasets that are helpful for learning Python and data analysis.

## Reflection (150–200 words)
Setting up these three platforms felt like building the first pages of a professional identity. Before this task, I had heard of LinkedIn and GitHub but had never actually created profiles on them. The process of filling out even the minimal details — degree name, institution, year — made me think ahead to what these profiles will look like in four years.

GitHub was the most technically new platform for me. Understanding that it is not just for storing code, but for demonstrating how you approach problems and collaborate with others, changed how I think about my coursework. I plan to upload every lab assignment and mini-project here from this semester onwards.

LinkedIn made me realise that professional networking is not something that begins after graduation — it begins now. I plan to connect with seniors from VIT Bhopal, follow companies I am interested in, and regularly update my profile as I gain skills and experiences.

Kaggle will be most useful in my second and third year when I begin exploring data science and AI. For now, I have completed my profile setup and explored the beginner Python course.
