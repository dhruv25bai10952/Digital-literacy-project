# Task 1 – Digital Literacy Infographic

## Tool Used
**Canva** (canva.com) – a free, browser-based graphic design platform.

## Design Description
The infographic is titled **"Be Digitally Literate: A Student's Guide"** and is structured as a single A4-sized visual slide with four clearly labelled sections:

1. **What is Digital Literacy?** – Defined as the ability to find, evaluate, create, and communicate information using digital technologies responsibly and effectively.
2. **Essential Digital Tools for Students** – Highlights platforms such as Google Workspace (Docs, Forms, Drive), GitHub for code management, and Canva for presentations.
3. **Safe Internet Practices** – Covers using strong passwords, enabling two-factor authentication, avoiding public Wi-Fi for sensitive tasks, and never sharing OTPs.
4. **Professional Online Presence** – Emphasises keeping social media profiles clean, using a college email for academic sign-ups, and maintaining a LinkedIn profile.

Colour scheme: Blue and white with accent icons for each section. Font: Poppins (sans-serif) for readability.

## Reflection (150–200 words)
To create this infographic, I used Canva, which provides an intuitive drag-and-drop design interface that requires no prior design experience. I began by selecting a blank A4 template and then built each section from scratch using text boxes, icons, and colour-coded panels.

My infographic covers three key areas of digital literacy: understanding what digital literacy actually means in the context of a student's life, the practical tools that can make academic work more efficient, and the safety habits every internet user must develop. I also included a short section on maintaining a professional online presence, since first impressions are now often made digitally.

The most challenging part was deciding which information to include and which to leave out, since there is far more to digital literacy than fits on a single page. I found it interesting to learn that "digital literacy" is not just about knowing how to use apps — it is about using them wisely, critically, and safely. This exercise made me realise how much of my own online behaviour I had never examined before.
